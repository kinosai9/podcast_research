初始化知识库

选择一个本地文件夹作为 Obsidian 知识库，系统会自动创建所需目录和基础文件。

{% if vault_path and not vault_missing %}
当前已配置路径: {{ vault_path }}
{% endif %}

你可以手动输入完整路径，或点击「选择文件夹」在资源管理器中选择。建议选择一个空文件夹。

初始化后会创建什么？

系统会在指定目录下创建以下结构：

  • 01_Reports/ — 研究报告
  • 02_Topics/ — 主题卡片
  • 03_Companies/ — 公司卡片
  • 05_Channels/ — 频道卡片
  • 06_Claims/ — 判断卡片
  • 07_Signals/ — 观察信号
  • 99_System/ — 系统索引和配置

如果目录非空，系统只会补齐缺失文件，不会删除已有内容。

初始化完成后，你可以在 Obsidian 中打开此目录作为 Vault。
